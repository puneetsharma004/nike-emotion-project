# 🏀 Nike Clone — React + Emotion JS

A modern Nike-inspired e-commerce UI clone built with **React (Vite)** and **Emotion JS**, 
focused on clean UI, scalable styling, and component-based architecture.

## 🚀 Tech Stack
- React (Vite)
- Emotion JS (CSS-in-JS)

## ✨ Features
- Responsive Nike-style layout
- Mega menu navigation
- Dynamic styling with Emotion
- Reusable, scalable component structure

## 📥 Getting Started
```bash
git clone -b setup <your-repo-url>
cd <your-repo-name>
npm install
npm run dev
```

> ⚠️ Make sure you're on the `setup` branch — the `main` branch is not fully configured.
